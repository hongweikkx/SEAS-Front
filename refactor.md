
# SEAS 学生成绩分析系统 - 页面重构任务书 (Final Version)

## 1. 项目愿景与视觉规范
- **项目名称：** SEAS (Student Evaluation & Analysis System)
- **目标用户：** 个人教师（C端）。强调**提效、AI辅助、专业且严谨**的办公体验。
- **核心视觉：** **现代学术风 (Modern Academic)**。
    - **配色：** 主色深藏蓝 (#1E3A8A) 用于侧边栏，智慧青 (#06B6D4) 用于高亮。
    - **质感：** 拒绝黑白灰。卡片背景采用浅彩色填充（如 `bg-blue-50/50`），并配合对应色系的淡色投影（Colored Shadows），增加视觉活力。
    - **AI 元素：** AI 相关组件使用淡紫色系 (#F5F3FF) 背景，并带有微弱的渐变流光或呼吸灯效果。

## 2. 页面全局布局
- **侧边栏 (Sidebar)：** 固定左侧，支持折叠。
    - **顶部：** 系统 Logo + SEAS 字样。
    - **中部：** 首页、分析列表、新建分析。
    - **底部：** 个人中心 / 系统设置 / 折叠切换按钮。
- **内容区 (Content)：** 保持大间距与呼吸感，使用模块化卡片布局。

## 3. 页面详细定义

### P1：登录页面 (Login)
- **功能：** 支持 **用户名密码** 与 **微信扫码** 切换。
- **视觉：** 左右分栏设计。左侧为唯美的学术意境图或功能介绍，右侧为简洁的登录操作卡片。

### P2：首页 (Dashboard)
- **欢迎区：** 专业的问候文案，如：“老师，欢迎回来。今天需要处理新的成绩数据吗？”
- **核心入口：** 醒目的 **“新建成绩分析”** 大按钮（视觉重心，使用渐变色增强引导感）。
- **AI 助手模块：** - **智能入口：** “想聊聊这届学生吗？” —— 点击进入 AI Chat 对话模式。
    - **免责声明：** 嵌入式提示条，告知 AI 分析结果仅供参考。
- **动态简报：** 展示最近一次分析的核心数据（如：及格率、平均分波动）。

### P3：分析列表 (Analysis List)
- **卡片式布局：** 每一项考试为一个独立卡片，包含：考试名称、日期、参与人数。
- **分类标签：** 用不同颜色标签区分：期中、期末、月考等。

### P4：新建分析 (Create Analysis)
- **交互：** 仪式感十足的拖拽上传区，支持 `.xlsx` / `.csv`。
- **引导：** 1. 文件投喂 -> 2. 字段校准 -> 3. AI 智能分析。

### P5：分析详情页 (Analysis Detail) - **核心重构区**
- **双段式侧边栏 (级联导航)：**
    - **上半部分 (科目切换)：** 采用 Tab 或 Pills 切换 [全科] 与 各个 [单科]。
    - **分割线：** 明显的视觉分隔。
    - **下半部分 (功能维度)：** 根据选定的科目动态加载按钮：
        - *全科视角：* 学科情况汇总、班级情况汇总、全科四率分析。
        - *单科视角：* 班级情况汇总、单科四率分析、临界生分析、成绩波动。
- **内容区 (模块化卡片布局)：**
    - **局部筛选配置：** 每个分析模块（如“四率分析卡片”）的标题栏右侧设有**独立配置入口**。
        - 用户可在卡片内直接调整“优秀线”、“及格线”或“临界分数”。
        - 修改后，**当前卡片**内的图表（饼图、柱状图）实时刷新，互不干扰。
    - **全局报告：** 页面顶部常驻 **“下载整体分析报告”** 按钮，根据当前所有模块的参数设定，一键生成汇总版 PDF/Word。
- **数据可视化：** 采用 ECharts/Recharts，配色需明快且具辨识度，严禁使用沉闷的灰色系。

## 4. 技术与交互细节
- **Tailwind 规范：** 禁止过度使用 Slate/Gray，多使用 Primary Color 的淡色变体（50/100档）。
- **微交互：** - 侧边栏折叠后悬浮显示文字。
    - 局部模块调整参数时，卡片内有局部 Loading 状态。
- **AI 增强：** 凡是 AI 生成的内容或洞察建议，侧面标注紫色 AI 标识。
- **空状态：** 无数据时显示定制的学术风格插画（如：整理试卷的老师）。
